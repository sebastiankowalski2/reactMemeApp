# React Meme Generator

A modern meme generator application built with React that allows users to create custom memes using popular meme templates from an external API.

## Features

- Browse popular meme templates from Imgflip API
- Add custom top and bottom text to memes
- Responsive design for mobile and desktop
- Fast and lightweight React application
- Built with Vite for optimal development experience

## Technologies Used

- **React** - JavaScript library for building user interfaces
- **Vite** - Next-generation frontend build tool
- **Imgflip API** - External API providing meme templates
- **CSS3** - Modern styling and responsive design

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/sebastiankowalski2/reactMemeApp.git
   cd react-meme-app
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint for code quality
